# furkot-sygic-itf

Generate Sygic ITF files from Furkot trip data.

## Install

```sh
$ npm install --save furkot-sygic-itf
```

## Usage

```js
import itf = from 'furkot-sygic-itf';

itf(ostream, trip);
```
