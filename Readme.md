# furkot-sygic-itf

Generate Sygic ITF files from Furkot trip data.

## Install

```sh
$ npm install --save furkot-sygic-itf
```

## Usage

```js
const itf = require('furkot-sygic-itf');

itf(ostream, trip);
```
